Given a 2D board containing 'X' and 'O' (the letter O), 
capture all regions surrounded by 'X'.

A region is captured by flipping all 'O's into 'X's in that surrounded region.
	
Input:

First line will have n,m the number of rows and the number of columns.
Each of the n line will m space seperated characters  'O' or 'X'.

Output:

Print the final surrounded region .

Constraints:

1 <= n,m <=5000
	
=========
Example 1
=========
Input
4 4
X X X X
X O O X
X X O X
X O X X

Output
X X X X
X X X X
X X X X
X O X X
