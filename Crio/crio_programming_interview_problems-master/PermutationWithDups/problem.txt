Compute all permutations of a string whose characters are not necessarily unique. The list of permutations should not have duplicates.

Input Format:
First and only line will contain a string s with lower alphabets.

Output Format:
Print all the possible unique permutations of the string.Each permutation should be in a new line.

Constraints:
1<=|s|<=10

=========
Example 1
=========

Input:

xrmm

Output:

xrmm
xmrm
xmmr
rxmm
rmxm
rmmx
mrxm
mrmx
mxrm
mxmr
mmxr
mmrx
