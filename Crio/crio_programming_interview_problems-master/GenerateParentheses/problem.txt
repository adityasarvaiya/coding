Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

INPUT
Single line containing one integer N denoting the number on pairs of parentheses allowed. 

CONSTRAINTS
1<=N<=12

OUTPUT
Combinations of valid sequence of N parentheses each in different line 

=========
Example 1:
=========

Input:
3

Output:
((()))
(()())
(())()
()(())
()()()

=========
Example 2:
=========

Input:
2

Output:
()()
(())
