Given a string, find out if string follows a given pattern or not.
If the string matches a pattern output Yes otherwise No.
The pattern must match a non empty string in the input string.
That is no letter of pattern string should be empty.

Input:
First line contains the number of test cases. For each test case you will be given two strings, i.e input string and pattern string. 

Output:
For each test case print Yes or No in a new line. Yes if pattern matches the Input string otherwise No.

Constraint:
1<=T<=50
1<=N<=50.
'a'<=s[i] <='z'.

==========
Example
==========
Input
1
graphtreesgraph aba

Output
Yes

Explaination:
Here if a-> graph , b-> trees .
So the input string matches the pattern. Hence Yes.
