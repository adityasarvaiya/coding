Given n points on a 2D plane, find the maximum number of points that lie on the same straight line.

INPUT
First line contains one number N denoting the number of points. 
Next N lines contain 2 numbers X, Y denoting X coordinate and Y coordinate of ith point. 

CONSTRAINTS
1<=N<=1000
-1000000<=X,Y<=1000000

OUTPUT
One number denoting the number denoting the maximum number of points that lie on the same straight line. 

=========
Example 1
=========
Input
3
1 1
2 2
3 3

Output
3

=========
Example 2
=========
Input
6
1 1
3 2
5 3
4 1
2 3
1 4

Output
4
