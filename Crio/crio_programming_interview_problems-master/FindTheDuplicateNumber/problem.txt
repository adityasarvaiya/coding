Given an array nums containing n + 1 integers,where each integer is between 1 and n (inclusive),
hence at least one duplicate number must exist. Assume that there is only one duplicate number, 
find the duplicate one.

=========
Example 1
=========
Input
5
2 4 5 3 1 5

Output
5

=========
Example 2
=========
Input
7
3 6 1 5 7 4 2 5

Output
5
