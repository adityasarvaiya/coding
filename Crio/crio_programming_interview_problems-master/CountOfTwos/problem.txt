Number of occurrences of 2 as a digit in numbers from 0 to n
Count the number of 2s as digit in all numbers from 0 to n.


Input:
 The first line consists of N.


Output:
A single integer which has the count of 2's.

Constraints:
1 ≤ N ≤ 10^6

==========
Example :
==========
Input:
22
Output :
6
Explanation:
 Total 2s that appear as digit from 0 to 22 are (2, 12, 20,21, 22);
