Given a non-empty string s and a dictionary wordDict containing a list of non-empty words,
determine if s can be segmented into a space-separated sequence of one or more dictionary words.

=========
Example 1
=========
Input
criodo
2
do
crio

Output
true

=========
Example 2
=========
Input
applepenapple
2
apple
pen

Output
true

=========
Example 3
=========
Input
catsandog
5
cats
dog
sand
and
dog

Output
false
